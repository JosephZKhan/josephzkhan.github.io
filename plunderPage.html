Plundermonium
June 2026



An online-multiplayer competitive collection game where players control boats by simultaneously spinning two wheels for steering and power.

This was made in a week for the Juniper Dev Serious Game Jam 2026. The given theme was 'Spin to Win', which we took in the direction of spinning a ship's steering wheel.

Using the open-source Mirror library for online multiplayer functionality in Unity, I created a lobby system for hosting and joining games.

I built the system for wave simulation using a Gerstner wave model. The Gerstner function is run multiple times for overlapping waves in different directions. The wave mathematics were taken from this tutorial on Catlike Coding's blog, though the code had to be translated for use in Unity 6's Shader Graph.

I also set up the UI for the rotating wheels, which are controlled by mouse click-and-drag and translate this rotational input to steering and forward movement.
